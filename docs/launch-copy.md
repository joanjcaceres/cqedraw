## Longer Post

cQEDraw is now available at https://cqedraw.org/

cQEDraw is an open-source web editor for superconducting circuit graphs. It
helps turn a circuit drawing into sparse capacitance and inverse-inductance
matrix snippets for Black Box Quantization workflows, preserves
Josephson-junction branch metadata, and can run initial modal analysis for
supported circuits directly in the browser.

The web app is the main interface: no local Python setup is needed to draw a
circuit, inspect generated matrices, or export an analysis table. It is designed
to pair with `sccircuits` when you want to continue the analysis in Python.

Try it here: https://cqedraw.org/
